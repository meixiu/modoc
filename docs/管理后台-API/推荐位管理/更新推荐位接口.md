    
**简要描述：** 

- 更新推荐位接口

**请求URL：** 
- ` http://domain/pos/update `
  
**请求方式：**
- POST 

**请求参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|posId |是  |int |推荐位ID   |
|alias |否  |string |推荐位别名   |
|desc     |是  |string |描述      |
|online |否  |string |上线时间   |
|enable |否  |string |状态 0:禁用 1:启用   |
|lang |否  |array |多语言   |

- Lang详解

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----    |
|lang_id  |是  |int    |语言Id    |
|title    |是  |string |标题      |

 **请求示例**
``` 
{
	"posId":1,
    "alias": "top",
	"desc": "desc"
    "online": "2018-11-11 11:11:11",
    "enable": 1,
    "lang": [{
		"lang_id": 1,
		"title": "title"
	}]
}
```
 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": []
}

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
