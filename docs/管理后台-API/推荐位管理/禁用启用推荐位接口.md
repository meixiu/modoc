    
**简要描述：** 

- 禁用启用推荐位接口

**请求URL：** 
- ` http://domain/pos/enable `
  
**请求方式：**
- POST 


**请求参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|posId |是  |array/int|推荐位Id   |
|enable |是  |int |状态 0:禁用 1:启用   |


 **请求示例**
``` 
{
    "posId": [1,2,3],
    "enable": 1
}
```
 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": []
}

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
