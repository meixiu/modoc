    
**简要描述：** 

- 广告删除接口

**请求URL：** 
- ` http://domain/ad/del `
  
**请求方式：**
- POST 

**请求参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|adId |是  |array |广告Id (array 批量操作）  |

 **请求示例**
``` 
{
	"adId":[1,2]
}
```
 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": []
}

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
