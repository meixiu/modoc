    
**简要描述：** 

- 广告信息接口

**请求URL：** 
- ` http://domain/ad/show `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|adId |是  |int |广告Id   |

|参数名|类型|说明|
