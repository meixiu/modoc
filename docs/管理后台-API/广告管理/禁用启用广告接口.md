    
**简要描述：** 

- 禁用启用广告接口

**请求URL：** 
- ` http://domain/ad/enable `
  
**请求方式：**
- POST 

**请求参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|adId |是  |int/array |广告Id (array 批量操作）  |
|enable |是  |int |语言Id   |

 **请求示例**
``` 
{
	"adId":[1,2],
    "enable": 1,
}
```
 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": []
}

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
