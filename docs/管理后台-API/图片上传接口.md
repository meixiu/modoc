    
**简要描述：** 

- 图片上传接口

**请求URL：** 
- ` http://domain/image/save `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|type |是  |string |广告：ad 封面：cover 缩略图：thumb   |

 **返回示例**

``` 
  {
    "code": 0,
    "msg": "success",
	"data": "路径"
  }
```
