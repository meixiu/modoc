    
**简要描述：** 

- 添加语料接口

**请求URL：** 
- ` http://domain/mui/create `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|identify |是  |string |标识   |
|message |是  |string | 原文    |
|lang     |是  |array | 内容    |

- Lang参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|language     |是  |string | 语言标识    |
|translation     |是  |string | 翻译    |


 **请求示例**
 
``` 
  {
    "identify": "identify",
    "message": "message"
	"lang": [
		{
			"language":"zh-cn",
			"translation":"消息"
		},
		{
			"language":"zh-zh",
			"translation":"消息"
		},
	]
  }
```

 **返回示例**

``` 
  {
    "code": 0,
	"msg":"success",
    "data": []
  }
```

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
