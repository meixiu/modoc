
    
**简要描述：** 

- 导入Excel接口

**请求URL：** 
- ` http://domain/backend/mui/excel `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|file |是  |file |excel表格   |


 **返回示例**

``` 
{
    "code": 1,
    "msg": "导入失败，请重试或联系管理员!",
    "data": []
}
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态码  |
|msg |string   |消息  |
|data |array   |数据  |

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
