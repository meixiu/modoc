    
**简要描述：** 

- 日志列表接口

**请求URL：** 
- ` http://domain/log/list `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|page |否  |int |页数   |
|pageSize |否  |int |页大小  |
|site_id |否  |int | 项目id    |
|author|否  |string |  操作者    |
|result|否  |int |  结果，1成功 0失败    |


 **请求示例**

```
{
    "page": 1,
	"page": 15,
    "site_id": 1,
    "author": 'xieshm',
	"result":1
}
```
