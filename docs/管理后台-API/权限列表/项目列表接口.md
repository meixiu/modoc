    
**简要描述：** 

- 【权限】项目列表接口

**请求URL：** 
- ` http://domain/perms/items`
  
**请求方式：**
- POST 

**参数：** 

无

 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": {
        "data": [
            {
                "ID": "4",
                "code": "1",
                "name": "女神2",
                "setting": null,
                "p": true
            }
        ],
        "admin": false,
        "time": 1529375882
    }
}
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态  |
|msg  |strint   |消息  |
|data |array   |数据  |

**DATA['data']数据字典**

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code    |string   | 项目ID  |
|name   |string   |项目名称  |
|...   |...   |...  |

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
