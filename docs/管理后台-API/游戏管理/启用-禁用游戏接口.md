    
**简要描述：** 

- 启用/禁用游戏接口

**请求URL：** 
- ` http://domain/games/enable`
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|gmId |是  |int |游戏Id（cms系统）   |
|enable |是  |int |启用:1 禁用:0   |

 **请求示例**
``` 
{
    "gmId":26,
	"enable":1
}
```
 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": []
}
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态  |
|msg  |strint   |消息  |
|data |array   |数据  |

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
