    
**简要描述：** 

- 启用/禁用语言接口（语言管理）

**请求URL：** 
- ` http://domain/lang/update`
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|lang |是  |array |项目Id   |

 **请求示例**
``` 
{
    "lang": [1,2,3],
	"enable": 1
}
```
 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": []
}
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态  |
|msg  |strint   |消息  |
|data |array   |数据  |

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
