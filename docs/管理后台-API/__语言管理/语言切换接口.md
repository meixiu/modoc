    
**简要描述：** 

- 语言切换接口

**请求URL：** 
- ` http://domain/lang/change`
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|langId |是  |int |语言Id   |


 **请求示例**
``` 
{
    "langId": 3
}
```
 **返回示例**

``` 
{
    "code": 0,
    "msg": "Sucess",
    "data": []
}
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |状态  |
|msg  |strint   |消息  |
|data |array   |数据  |

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
