    
**简要描述：** 

- 语言列表接口

**请求URL：** 
- ` http://domain/lang/list `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|valid  |否  |int |1 启用的语言列表，不传或0全部列表   |

|lang_name       |array       |	语言名称   |
|enable       |array       |	状态 1:启用 0:禁用   |
