    
**简要描述：** 

- 删除接口

**请求URL：** 
- ` http://xx.com/article/del `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|artId |是  |int/array |文章Id（批量操作传id数组）   |

 **请求示例**
``` 
{
	"artId": [1,2],
}

```
 **返回示例**

``` 
{
	"code": 0,
	"msg":"success",
	"data": []
}
```

 **备注** 

- 更多返回错误代码请看首页的错误代码描述
