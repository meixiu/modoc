    
**简要描述：** 

- Ajax分页数据接口

**请求URL：** 
- ` http://domain/{lang}/{cate}/viewall?page={num} `
  
**请求方式：**
- get 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|page |是  |int |页数   |
|cateId |是  |int |栏目ID   |


 **请求示例**

```
/en/news/viewall?page=2
```
